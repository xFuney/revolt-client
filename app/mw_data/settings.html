SETTINGS

Rudimentary Developer Settings (not on brand/style)

Linux Window Manager Toggle

If enabled, the client will use its custom top bar. If not, the Window Manager's frame will be used.

Requires a restart of the client to be applied.

Minimum Window Size Toggle

If enabled, the client will enforce a minimum window size. Otherwise, no minimum window size
will be enforced. Requires a restart of the client to be applied.

Top Bar Hexadecimal Value

Specifies the color for the top bar (for Linux users, only when using the custom top bar).
